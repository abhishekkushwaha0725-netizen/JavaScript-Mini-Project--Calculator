# 🧮 JavaScript Calculator App

A simple and interactive calculator web application built using **HTML, CSS, and JavaScript**.  
It performs basic arithmetic operations and demonstrates DOM manipulation skills.

---

## 🚀 Features
- Basic arithmetic operations (+, -, *, /)
- Clear (C) and Backspace functionality
- Keyboard support (numbers, operators, Enter, Backspace)
- Dark / Light mode toggle
- Clean and responsive UI
- Real-time calculations using JavaScript

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)
